# didala
基于 TypeScript + React + Hooks + RxJs + Nodejs + MongoDB 实现的 仿滴答清单的全栈应用

### Demo


<img src="https://buynao.oss-cn-beijing.aliyuncs.com/20200406_193852.gif"/>

[http://dida.buynao.com](http://dida.buynao.com)

### 本地启动

```bash
# 安装前端所需要的依赖包
npm i

# 启动前端服务，监听本地1026端口
npm run start
```
